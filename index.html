¿Qué es EneasOS?

Es un sistema operativo creado específicamente para desplegar aplicaciones web básicas de estructura HTML, CSS y JS. Es un sistema basado en la librería de Circle para Raspberry Pi 4 de 64 bits.

Global

Tiene una utilidad en cualquier parte del mundo

Seguridad

Servidor Portable

Tiene todas las cualidades de un servidor web y es mejor debido a su portabilidad
